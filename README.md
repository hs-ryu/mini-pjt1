# Way (Where are you)

**내가 갔던 곳의 기록을 남기고 싶다면?**

> 내가 기록을 남긴 지역들을 한눈에 볼 수 있는 서비스.
>
> 사진과 글을 업로드 할 수 있어요.
>
> 다양한 장소에서 기록을 남겨보세요.

<br>

## Stack

1. 이슈관리 : Miro
2. 개발환경
   - OS : MacOS (M1Pro)
   - IDE : Xcode 13.3
3. 상세 사용
   - Application : SwiftUI
   - Design : Sketch

<br>

## Features

- 위치 정보 수집
- 휴대폰 앨범 접근
- 사진과 글로 기록 남기기
- 내가 남긴 기록들을 한눈에 보기
